# DrawerSidebar

DrawerSidebar is typically used for navigation. It is initially hidden off-screen but can be pulled in from the edge of the window. The drawer's open state is controlled through a scoped model via the `$open` prop, and its content is provided through the `renderContent` render prop.

```jsx
import { DrawerSidebar } from 'startupjs-ui'
```

## Simple example

```jsx example
const $open = $()
return (
  <Div style={{overflow: 'hidden'}}>
    <DrawerSidebar
      $open={$open}
      renderContent={() => (
        <Span>
          DrawerSidebar
        </Span>)
      }
    >
      <Button onPress={() => $open.set(true)}>Open</Button>
    </DrawerSidebar>
  </Div>
)
```

## Position

The `position` prop specifies which side of the screen the drawer slides in from. Possible values are `left` (default) and `right`.

```jsx example noscroll
const $open = $()
return (
  <Div style={{overflow: 'hidden'}}>
    <DrawerSidebar
      $open={$open}
      position='right'
      renderContent={() => (
        <Menu>
          <Menu.Item>Nav-1</Menu.Item>
          <Menu.Item>Nav-2</Menu.Item>
          <Menu.Item>Nav-3</Menu.Item>
          <Menu.Item>Nav-4</Menu.Item>
        </Menu>
      )}
    >
      <Content
        padding
        width='full'
        style={{ backgroundColor: 'white' }}
      >
        <Div row>
          <Button onPress={() => $open.set(true)}>Open</Button>
        </Div>
        <Br />
        <Span>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Rhoncus dolor purus non enim praesent elementum facilisis leo vel. Risus at ultrices mi tempus imperdiet. Semper risus in hendrerit gravida rutrum quisque non tellus.
        </Span>
      </Content>
    </DrawerSidebar>

)
```

## Lazy rendering

By default, the drawer content is not destroyed when the drawer is closed. Set `lazy` to `true` to render the drawer content only while it is open, which can improve performance for heavy sidebar content.

```jsx example
const $open = $()
return (
  <Div style={{overflow: 'hidden'}}>
    <DrawerSidebar
      $open={$open}
      renderContent={() => (
        <Span>
          Lazy DrawerSidebar
        </Span>)
      }
      lazy
    >
      <Button onPress={() => $open.set(true)}>Open</Button>
    </DrawerSidebar>
  </Div>
)
```

## Additional props

- **style** -- custom styles applied to the root container
- **children** -- the main page content rendered alongside the drawer
- **$open** -- a scoped model that controls whether the drawer is open or closed
- **width** -- the drawer width in density-independent pixels (default `264`)
- **disabled** -- set to `true` to lock the drawer closed and prevent user interaction
- **renderContent** -- a function that returns the content to render inside the drawer panel
