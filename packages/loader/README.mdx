# Loader

Displays a spinning activity indicator to signal that content is loading or a background process is in progress.

```jsx
import { Loader } from 'startupjs-ui'
```

## Simple example

Render a default loader with no additional configuration.

```jsx example
return (
  <Loader />
)
```

## Sizes

The `size` prop controls the diameter of the spinner. Available values are `'s'` (small) and `'m'` (large). The default is `'m'`.

```jsx example
return (
  <Div row align='around'>
    <Loader size='s' />
    <Loader size='m' />
  </Div>
)
```

## Colors

By default the loader uses the `'text-description'` color token. You can change it by passing any color name from the theme's color palette to the `color` prop. See the Sandbox section below for all available values.

```jsx example
return (
  <Div row align='around'>
    <Loader />
    <Loader color='success' />
    <Loader color='error' />
    <Loader color='warning' />
  </Div>
)
```
