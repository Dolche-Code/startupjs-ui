# FlatList

A performant list component for rendering large scrollable datasets. Re-exported directly from [React Native](https://reactnative.dev/docs/flatlist) and [React Native Web](https://necolas.github.io/react-native-web/docs/lists/). Refer to those docs for the full list of available props.

```jsx
import { FlatList } from 'startupjs-ui'
```
