# Span

The base component of typography is `Span`, which is used for plain text.

```jsx
import { Span } from 'startupjs-ui'
```

## Simple example

```jsx example
<Span>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Span>
```

## Headers

Use `h1` - `h6` props to create headers.

```jsx example
<Span h1>H1 Heading</Span>
<Span h2>H2 Heading</Span>
<Span h3>H3 Heading</Span>
<Span h4>H4 Heading</Span>
<Span h5>H5 Heading</Span>
<Span h6>H6 Heading</Span>
```

## Bold and Italic

```jsx example
<Span bold>Bold text</Span>
<Span italic>Italic text</Span>
<Span bold italic>Bold and Italic text</Span>
```

## Description

Use `description` prop for secondary text.

```jsx example
<Span description>Description text</Span>
```

## Animations

Div automatically uses React Native Reanimated when a `transition` or `animation` property is detected in the style. This allows you to create smooth animations without any additional setup.

**Note:** When using modifier classes to add transitions/animations dynamically, set `animation: none` in the base styles. This ensures Div uses the animated component from the start.

```jsx example
const [isActive, setIsActive] = useState(false)
return (
  <View styleName='root'>
    <Span styleName={['text', { isActive }]}>Hello World!</Span>
    <Pressable styleName='button' onPress={() => setIsActive(!isActive)}>
      <Span style={{ color: 'white' }}>Activate</Span>
    </Pressable>
  </View>
)
styl`
  .text
    color red
    transition color 1s
    animation none
    &.isActive
      color green
      animation shake 0.5s infinite, pulse 1s infinite, highlight 1s infinite
  .root
    height 60px
    flex-direction row
    align-items center
    gap 16px
  .button
    padding 8px 16px
    background-color #2962FF
    border-radius 4px
  @keyframes shake
    0%, 100%
      transform translateX(0) translateY(0)
    25%
      transform translateX(-10px) translateY(-10px)
    75%
      transform translateX(10px) translateY(10px)
  @keyframes pulse
    50%
      font-size 24px
  @keyframes highlight
    50%
      background-color rgba(green, 0.2)
      box-shadow 0px 0px 10px rgba(green, 1)
`
```
